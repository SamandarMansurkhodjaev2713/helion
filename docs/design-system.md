# Дизайн-система

Настроение: **Interstellar / Dune** — холодный кинематографичный космос: масштаб,
тишина, сдержанность. Глубокий тёмно-синий фон, один ледяной циановый акцент,
приглушённый стальной для телеметрии, зерно и виньетка для «плёночности».

## Палитра

Токены заданы в двух местах и обязаны совпадать: `tailwind.config.js` (утилиты-классы) и
`src/index.css` (CSS-переменные для canvas и imperative-кода).

| Токен | HEX | Роль |
| --- | --- | --- |
| `void` | `#060910` | Фон, глубокий космос |
| `deep` | `#0B1120` | Крупные панели, второй слой |
| `panel` | `#111A2B` | Карточки, приподнятые поверхности |
| `accent` | `#6FD3F2` | **Главный** ледяной акцент: CTA, хайлайты |
| `accent-bright` | `#A9E7FF` | Hover / свечение |
| `accent-deep` | `#3E9FC4` | Нажатие / тень |
| `steel` | `#6E8BA6` | Приглушённый вторичный: телеметрия |
| `ice` | `#9DBBD6` | Мягкий холодный хайлайт |
| `bone` | `#ECE6DA` | Основной текст |
| `ash` | `#8B8578` | Мета, подписи |

Правило: **1–2 циановых акцента на экран**. Steel — скупо, для служебного слоя.
Прозрачности задаются модификатором Tailwind (`text-bone/60`).

## Типографика

| Роль | Семейство | Класс | Применение |
| --- | --- | --- | --- |
| Display | Unbounded (500–700) | `font-display` | Акцент-строки заголовков, вордмарк, имена/названия |
| Body | Inter | `font-sans` | Весь основной текст |
| Mono | JetBrains Mono | `font-mono` | Телеметрия, eyebrows, теги, **все цифры/статы** |

- Сигнатура заголовка секции: первая строка — `font-light` Inter; вторая строка —
  `font-display uppercase text-accent` в ~0.52em от размера первой (Unbounded шире,
  оптическая компенсация обязательна).
- Названия (миссии, борта, имена, вехи): `font-display font-medium`, `text-base…text-2xl`.
- Числа и характеристики: `font-mono font-medium tabular-nums`.
- Eyebrows: `font-mono`, `text-[11px]`, `uppercase`, `tracking-telemetry` (0.28em),
  `text-steel`.
- Body: `text-base…text-lg`, `leading-relaxed`, `text-bone/60`.

## Пространство и форма

- Секции: `max-w-7xl`, паддинги `px-6 md:px-10`, `py-28 md:py-40`.
- Скругления: карточки `rounded-2xl`, пилюли/кнопки `rounded-full`.
- Границы: `border-white/10` по умолчанию; hover — `border-accent/30`.
- Разделители: класс `.hairline` (градиентная линия), короткие `h-px w-12 bg-white/20`.

## Текстуры

- `.grain` — плёночное зерно (SVG-шум, `mix-blend-mode: overlay`), на хиро и портретах.
- `.vignette` — радиальное затемнение краёв кадра.
- `.hud-lines` — тонкие скан-линии для HUD-поверхностей.
- Атмосферные градиенты секций — `SectionShell` (`atmosphere="accent|steel"`), медленный
  параллакс-дрейф.

## Движение

- Длительности: 500–1200 мс. Easing: `cubic-bezier(0.16, 1, 0.3, 1)`.
- Классы: `ease-cinematic`, `duration-{600,800,1000,1200}`.
- Тюнинг imperative-движения — объект `MOTION` в `src/lib/constants.ts`.
- Полное уважение `prefers-reduced-motion` (см. `animations.md`).

## Чек-лист применения

- Цвет только из токенов; исключение — rgba в градиентах/тенях (пометить «mirrors --accent»).
- Не более 1–2 циановых акцентов на экран.
- Mono — только для служебной информации, не для основного чтения.
- Любая новая поверхность проверяется на 375px и 1280px.
